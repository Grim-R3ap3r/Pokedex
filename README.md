# Pokedex
Pokedex made using python

Packages required-->

->pip install pypokedex

->pip install utllib3

->pip install pillow



About:

Using the pypoedex library I have made a pokedex.

Pokedex is a device used in pokemon to identify a pokemon and tell all the details of that pokemon.
